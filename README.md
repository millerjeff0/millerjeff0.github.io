OcarinaTrainer
       
A practice companion for 12-hole soprano-C transverse ocarina players. Displays fingering charts, listens to your playing, and provides real-time pitch feedback.
      
## Features
       
- Fingering charts for 20 chromatic notes (A4–F6)
- Real-time pitch detection with built-in tuner
- Arc-style pitch meter showing cents deviation
- C Major and Chromatic scale modes
- Adjustable listening sensitivity
